# 指南

欢迎来到 AI 编程教程！本指南将帮助你系统学习现代 AI 编程的核心范式。

## 教程结构

本教程采用 [Diátaxis 框架](https://diataxis.fr/) 组织内容，分为四类：

### 📖 Tutorials（教程）
学习导向，手把手教学。适合初学者从零开始。

- [创建你的第一个 Skill](/guide/skills/first-skill)
- [OpenSpec 完整工作流](/guide/openspec/workflow)
- [编写第一个测试场景](/guide/harness/writing-tests)

### 🛠️ How-to Guides（指南）
目标导向，解决具体问题。适合有明确目标的开发者。

- [Skill 命令参考](/guide/openspec/commands)
- [Evaluators 使用指南](/guide/harness/evaluators)
- [Mock Server 配置](/guide/harness/mock-server)

### 📚 Reference（参考）
信息导向，命令/配置参考。适合快速查阅。

- [命令快速参考](/reference/commands)
- [FAQ](/reference/faq)
- [故障排除](/reference/troubleshooting)

### 💡 Explanation（解释）
理解导向，概念和原理。适合深入理解。

- [什么是 Skill](/guide/skills/what-is-skill)
- [OpenSpec 核心概念](/guide/openspec/concepts)

## 学习建议

1. **初学者**：从 Tutorials 开始，跟着教程动手实践
2. **有经验的开发者**：直接跳到 How-to Guides，按需学习
3. **架构师**：重点看 Explanation 部分，理解设计原理
4. **所有人**：Reference 随时查阅，不必记忆
