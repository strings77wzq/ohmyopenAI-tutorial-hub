# 规格：Emoji 替换标准

## 替换规则

### 对错标记
- ✅ → (✓) 正确 / [正确]
- ❌ → (✗) 错误 / [错误]

### 优先级标记
- 🔴 → 【高】/ [P0-高优先级]
- 🟡 → 【中】/ [P1-中优先级]
- 🟢 → 【低】/ [P2-低优先级]

### 模块图标
- 🧠 → [技能] / [Skill]
- 📋 → [规范] / [Spec]
- 🧪 → [测试] / [Test]
- 🚀 → [实战] / [Project]
- ⚡ → [快速] / [Quick]

### 角色图标
- 👨‍💻 → 开发者 / Developer
- 🏗️ → 架构师 / Architect
- 🎓 → 学习者 / Student

### 其他图标
- ⭐ → 文字描述（如"使用频率：高"）
- 📐 → [质量] / Quality
- 🔒 → [安全] / Security

## 实施要求

1. 所有 emoji 必须替换为文字或标记
2. 保持语义清晰，易于理解
3. 在同一文件中保持一致的替换风格
4. 替换后检查可读性
