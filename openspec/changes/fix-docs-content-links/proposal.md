## Why

官网首页存在内容问题和定位错误：

1. OMO模块缺少GitHub repo链接，用户无法深入了解项目
2. 与Claude Code对比的描述不专业，显得外行
3. golem案例介绍尴尬，未体现真正价值，且缺少项目推广链接

影响：降低网站专业度，无法达成"通过教程推广自己的项目"的目标。

## What Changes

1. **添加OMO模块GitHub链接** - 添加Oh-My-OpenAgent repo链接
2. **删除与Claude Code对比的内容** - 这部分描述不专业
3. **重写golem案例介绍** - 体现真实价值，并添加golem项目GitHub链接
4. **优化Feature卡片内容** - 更专业的描述
