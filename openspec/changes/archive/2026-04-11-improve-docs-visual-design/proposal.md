## Why

当前站点缺少视觉吸引力和专业感，对比 oh-my-openagent、oh-my-claudecode、oh-my-codex、hermesagent 等竞争站点：
1. 首页不够吸引人，缺少核心卖点和统计数据
2. 缺少 What's New/Changelog 板块
3. 没有明确的学习路径导航
4. 缺少用户评价/社会证明（即使是"期待您的第一条评价"也要诚实展示）
5. 导航结构不够清晰

## What Changes

- 更新首页 Hero 区，增加统计数据和标语
- 添加 What's New 版本更新日志板块
- 改进 Features 展示卡片
- 添加学习路径导航
- 更新站点 title 和描述

## Capabilities

### Modified Capabilities

- `docs-homepage`: 首页视觉和内容优化
- `docs-navigation`: 清晰的导航结构
